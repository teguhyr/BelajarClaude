# Panduan Training Claude - Index Modul

**Belajar Bareng Claude untuk HR & Productivity Professional**

---

## Tentang Training Ini

Training ini dirancang untuk membantu HR professional dan knowledge worker memaksimalkan penggunaan Claude AI dalam pekerjaan sehari-hari. Materi mencakup dua level dengan fokus khusus pada fitur **Co-work** (kolaborasi tim) dan **Code** (otomasi dan data).

**Target Peserta:**
- HR Manager, HR Generalist, HR Business Partner
- Tim Talent Acquisition dan Learning & Development
- Knowledge worker yang ingin meningkatkan produktivitas dengan AI

---

## Struktur Training

```
Level Basic (Modul B01-B04)
```

## Konten Setiap Level

### Level Basic - Apa yang Akan Kamu Pelajari

**B01 - Mengenal Claude:**
- Apa itu Claude dan siapa pembuatnya (Anthropic)
- Perbedaan Claude vs ChatGPT vs Gemini
- Cara akses: claude.ai web dan Claude Code CLI
- Batasan Claude yang penting diketahui
- Etika penggunaan AI di tempat kerja

**B02 - Setup dan Antarmuka:**
- Cara membuat akun di claude.ai
- Mengenal setiap bagian antarmuka
- Pilih model yang tepat: Haiku, Sonnet, Opus
- Keyboard shortcuts dan tips navigasi
- Cara upload file ke Claude

**B03 - Prompt yang Efektif:**
- Anatomi prompt: Konteks + Tugas + Format + Batasan
- Kesalahan umum dan cara menghindarinya
- Teknik role prompting dan few-shot prompting
- 10 template prompt siap pakai untuk HR
- Latihan memperbaiki prompt yang buruk

**B04 - Claude untuk HR:**
- Job description, pertanyaan interview, email HR
- Meringkas dokumen kebijakan panjang
- Membuat materi training
- Analisis data survei karyawan (anonimkan dulu)
- Workflow rekrutmen end-to-end dengan Claude
- Do's and Don'ts privasi data karyawan

---

### Level Intermediate Co-work Track - Apa yang Akan Kamu Pelajari

**I01 - Projects:**
- Perbedaan Projects vs chat biasa
- Setup Project baru dengan Project Instructions
- Upload dokumen ke Knowledge base project
- Berbagi Project dengan anggota tim
- Strategi organisasi project untuk HR

**I02 - Artifacts:**
- Jenis Artifacts: teks, kode, diagram, HTML
- Membuat dan mengedit dokumen sebagai Artifact
- Diagram alur proses dengan Mermaid (flowchart, Gantt)
- Share dan export Artifacts
- Praktik: buat SOP HR sebagai Artifact

**I03 - Team Workflows:**
- Share conversation ke rekan tim
- Template prompt standar untuk pekerjaan berulang
- Integrasi Claude dengan Notion, email, spreadsheet
- Delegasi riset ke Claude
- Governance penggunaan AI di tim
- Contoh lengkap: persiapan Town Hall Meeting

---

### Level Intermediate Code Track - Apa yang Akan Kamu Pelajari

**I04 - Code Dasar:**
- Kenapa HR professional perlu fitur code
- Formula Excel/Google Sheets: SUMIF, VLOOKUP, IF bertingkat
- Script Python untuk memproses data CSV
- Query SQL dasar untuk database HRIS
- Prinsip keamanan sebelum menjalankan kode

**I05 - Debugging & Code Review:**
- Apa itu debugging (penjelasan non-teknis)
- Template request debug yang efektif
- Common errors Excel dan cara memperbaikinya
- Code review: minta Claude periksa logika dan keamanan
- Keamanan data saat debugging - apa yang tidak boleh di-paste

**I06 - Claude Code CLI:**
- Perbedaan Claude Code CLI vs claude.ai web
- Instalasi di macOS, Windows, Linux
- Perintah dasar interaktif dan one-shot
- Membaca dan mengedit file langsung dari terminal
- Analisis folder project secara keseluruhan
- Keamanan dan permission Claude Code

## Prinsip Umum Penggunaan Claude

**Yang selalu berlaku di semua modul:**

1. **Claude bisa salah** - selalu verifikasi informasi penting sebelum digunakan
2. **Review sebelum publish** - semua output Claude perlu di-review manusia sebelum digunakan resmi
3. **Privasi data** - jangan paste data karyawan yang mengidentifikasi individu
4. **Iterasi** - hasil pertama jarang sempurna, gunakan percakapan lanjutan untuk perbaikan
5. **Konteks itu penting** - semakin detail konteks yang kamu berikan, semakin baik hasilnya
